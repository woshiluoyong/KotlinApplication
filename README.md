# KotlinNcpApp
***
本应用采用kotlin语言编写查看NCP(新型冠状肺炎)疫情数据，应用到其中的协程实现线程切换，采用MVP模式实现，还用到了Anko、RxJava、Retrofit、MaterialDesign相关技术
